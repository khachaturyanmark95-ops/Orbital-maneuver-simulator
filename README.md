# Orbital Maneuver Simulator

An animated simulation of the Hohmann Transfer Orbit using real orbital mechanics.

## What is a Hohmann Transfer?

A Hohmann Transfer is the most fuel-efficient way to move a satellite between two circular orbits. It uses exactly two engine burns.

## Orbits simulated

- LEO (Low Earth Orbit) - 400 km altitude (where ISS flies)
- GEO (Geostationary Orbit) - 35,786 km altitude (where communication satellites are)

## Physics

- Vis-viva equation for orbital velocities
- Delta-V calculations for both burns
- Real gravitational constant and Earth mass

## Technologies

- Python 3
- NumPy
- Matplotlib
